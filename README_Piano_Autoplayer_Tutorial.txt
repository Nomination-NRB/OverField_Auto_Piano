Windows 钢琴自动演奏 - 乐谱编写教程 (LRC 风格 .lrcp)

一、键位与音符标记
- 低音：L1-L7  -> 键 a s d f g h j
- 中音：M1-M7  -> 键 q w e r t y u
- 高音：H1-H7  -> 键 1 2 3 4 5 6 7
- 和弦：C Dm Em F G Am G7 -> 键 z x c v b n m

二、行格式（类似 LRC）
[mm:ss.xxx] TOKEN1 TOKEN2 TOKEN3 ...
- 时间戳 [分:秒.毫秒]，毫秒可省略；同一行可写多个时间戳（与 LRC 一致）
- 同一时间戳后面的多个 TOKEN 表示“同时触发”（和弦/多音）
- TOKEN 必须是：
    - L1-L7（低音）/ M1-M7（中音）/ H1-H7（高音）
    - 或和弦名：C/Dm/Em/F/G/Am/G7

示例：
[00:01.200] M1 M3        # 在 1.200s 同时按中音1与中音3
[00:02.000][00:03.000] C # 在 2.000s 与 3.000s 各弹一次 C 和弦

三、节奏与速度
- 乐谱时间是“真实秒”，若想整体加速/减速，使用程序中的“速度比例”而非修改每个时间戳。
  例如原速 1.0；2.0 表示加倍速度（事件时间轴等比例压缩）。

四、常见写法
1) 单旋律：每个音写在相邻的时间戳上：
   [00:00.000] M1
   [00:00.600] M1
   [00:01.200] M5
   …（每 600ms 一个四分音符，100BPM）

2) 加入和弦：在同一时间戳加一个和弦 token：
   [00:00.000] M1 C

3) 多音同时：直接并列多个 token：
   [00:01.800] M3 M5 M1

五、对齐与延迟
- 如果游戏有输入延迟，可在程序里设置“全局延迟（毫秒）”。
- 演奏前会有“起始倒计时”，便于切换回游戏并聚焦窗口。

六、文件扩展名
- 建议使用 .lrcp（LRC for Piano），普通 .txt 也可。编码使用 UTF-8。
